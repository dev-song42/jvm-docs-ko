# 문서 템플릿

새 문서는 로드맵에서 다음 주제를 선택한 뒤 이 디렉터리의 템플릿을 복사해 시작합니다.

## 템플릿 선택

- `study-note.mdx`: Java처럼 공식 자료를 참고해 직접 공부하고 설명하는 문서
- `translation.mdx`: Spring Framework와 Spring Boot 공식 문서를 선별해 번역하는 문서

## 상태 흐름

1. **작성 전**: 공개 로드맵에만 있으며 실제 문서 파일은 만들지 않습니다.
2. **작성 중**: 템플릿을 `docs/` 아래에 복사하고 `draft: true`로 작성합니다.
3. **작성 완료**: 문서를 검토한 뒤 `draft: true`를 제거하고 로드맵에 문서 링크를 연결합니다.

작성 중인 주제를 공개 로드맵에도 표시하려면 로드맵 상태 변경을 먼저 병합합니다. 짧은 문서는 문서 완성과 함께 바로 `작성 완료`로 변경해도 됩니다.

## 작성 규칙

- 파일명과 경로는 소문자 kebab-case를 사용합니다.
- 페이지마다 기준 버전과 주요 출처를 기록합니다.
- 직접 작성한 해설과 번역문을 문서 유형으로 명확하게 구분합니다.
- 번역문은 원문의 제목 구조를 따르고, 추가 설명은 `TranslatorNote`로 구분합니다.
- 코드 예제는 직접 검증한 최소 예제를 우선합니다.
- 원문의 라이선스와 재배포 조건을 문서별로 확인합니다.

## 브랜치 예시

```text
docs/java/01-java-platform
docs/spring/01-ioc-container
docs/boot/04-auto-configuration
```
