## 추가 원칙

Java라는 이름 아래에는 Oracle Java 문서, OpenJDK 프로젝트 문서, dev.java 학습 자료처럼 저작권과 라이선스가 서로 다른 자료가 존재합니다. 따라서 하나의 라이선스로 묶어서 가져오지 않고 문서별 출처와 허용 범위를 먼저 기록합니다.

## 후보 주제

- 언어 기본 문법보다 실무에서 자주 혼동하는 핵심 개념
- 컬렉션과 스트림의 올바른 사용법
- 동시성 기초
- JVM 동작을 이해하는 데 필요한 공식 자료
- 최신 LTS 버전으로 이동할 때 필요한 변경 사항

현재는 Spring Framework와 Spring Boot 문서 구조를 먼저 안정화합니다.
