## Java부터 시작하는 이유

Spring의 의존성 주입, 프록시, 트랜잭션을 제대로 이해하려면 Java의 타입 시스템, 애노테이션, Reflection, 예외, 동시성, JVM 실행 구조가 먼저 필요합니다.

## 작성 원칙

Java라는 이름 아래에는 Oracle Java 문서, OpenJDK 프로젝트 문서, dev.java 학습 자료처럼 저작권과 라이선스가 서로 다른 자료가 존재합니다. 따라서 하나의 라이선스로 묶어서 가져오지 않고 문서별 출처와 허용 범위를 먼저 기록합니다.

기준 예제는 Java 21로 작성하고, Preview·Incubator 기능과 전체 API 사전은 첫 범위에서 제외합니다.
