## 로드맵 갱신 규칙

1. 작업 브랜치를 만들 때 대상 주제를 선택합니다.
2. 템플릿으로 초안을 만들고 필요하면 상태를 `작성 중`으로 변경합니다.
3. 검토와 빌드를 마치면 `draft: true`를 제거합니다.
4. 상태를 `작성 완료`로 바꾸고 실제 문서 링크를 연결합니다.
5. 원문 버전이 바뀌면 해당 문서의 기준 버전과 최근 검토일을 갱신합니다.
