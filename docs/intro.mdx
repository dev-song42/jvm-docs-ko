## 문서를 읽는 방법

각 문서 상단의 정보 카드에서 다음 내용을 확인할 수 있습니다.

- 공식 원문 링크
- 번역 기준 버전
- 현재 번역 및 검토 상태
- 마지막 검토 날짜

원문에는 없지만 이해를 돕기 위해 추가한 설명은 **번역자 노트**로 명확하게 구분합니다.

## 현재 진행 순서

1. Spring Framework 6.2의 핵심 컨테이너 개념
2. Spring Boot 3.5.6의 핵심 개발·운영 기능
3. Java 공식 문서 중 재배포 가능한 중요 주제

왼쪽 목차에서 문서를 선택하거나, 위 카드에서 프로젝트별 첫 페이지로 이동할 수 있습니다.
