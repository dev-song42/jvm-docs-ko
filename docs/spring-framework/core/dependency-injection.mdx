## 다룰 내용

- 의존성 주입이 해결하는 문제
- 생성자 기반 의존성 주입
- 세터 기반 의존성 주입
- 순환 의존성과 조기 오류 발견
- 생성자 주입을 우선할 때의 장점

이 페이지는 문서 뼈대를 위한 자리표시자입니다. 번역을 시작하면 원문 기준 커밋과 검토 상태를 갱신합니다.
