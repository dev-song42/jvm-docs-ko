## 왜 Framework부터 읽나

Spring Boot는 Spring Framework 위에 편리한 기본값과 자동 구성을 더합니다. 따라서 IoC 컨테이너, 빈, 의존성 주입 같은 기반 개념을 먼저 이해하면 Boot의 자동 동작도 훨씬 명확하게 보입니다.
